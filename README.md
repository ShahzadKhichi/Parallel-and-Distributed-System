# PDC Assignment - Mesh Circular Shift Visualizer

This is a comprehensive React + TypeScript application that visualizes Parallel and Distributed Computing concepts.

## Features
- **Mesh Shift**: Interactive $4 \times 4$ to $8 \times 8$ toroidal grid with circular shift animations.
- **Topologies**: SVG representations of Ring, Mesh, and Hypercube networks.
- **Hypercube All-to-All**: Step-by-step bit-ordered exchange simulation.
- **Premium UI**: Built with Tailwind CSS, Framer Motion, and Lucide icons.

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started
```bash
npm install
npm run dev
```
